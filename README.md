# jnulib-auto-save-book
Automatically save images into pdf book for jnu library.
##使用说明
（访问 http://域名或IP以及目录路径/auto-save-book-v*.php）

把【存书】链接拖到书签栏，在图书馆书籍页面【点击阅读电子版】打开的页面上点击一下该书签，耐心等待足够长的时间，右上角会弹出生成pdf的对话框。
注意：该版本不支持多个相同任务同时执行。
